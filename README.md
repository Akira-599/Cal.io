# Cal. — Indian Holiday Calendar

A beautiful, minimal Indian holiday calendar app.

## How to get the APK

1. Upload `calendar.html` to the `www/` folder in this repo (rename it to `index.html`)
2. GitHub will automatically build the APK
3. Go to the **Actions** tab → click the latest build → scroll down to **Artifacts** → download **Cal-app**
4. Extract the zip, install the `.apk` on your Android phone

## Installing on Android

1. On your phone go to **Settings → Security → Unknown Sources** and enable it
2. Copy the APK to your phone and tap it to install

Optimized for Android including Redmi 8A Dual (Snapdragon 439).
